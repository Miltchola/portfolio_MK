Gamer Moment (GM)

A Digital Games API focused on dynamism and building a Video Game Review community.

About the Project

This project, developed during the Full Stack course, aimed to create a digital games API focused on dynamism and building a video game review community. The team developed this project using React.js and TypeScript.

The API is still under production, therefore, the image presented does not correspond to the final version of the website, but rather a reference from another API used as a development base.
