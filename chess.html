A Java recreation of the classic board game, Chess.

About the Project

A Recreation of the Classic Boardgame, Chess, but it was enterely made using Java and JavaSwing.

This Project was made in my fifth Semester, during P.O.O. (Object-Oriented Orogramming) Subject in SENAI CIMATEC University Center. The idea was to recreate the boardgame throught the course, but utilizing our knowledge on heritage, encapsulation and graphical interface.
