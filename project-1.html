About the Project

A Challenge made for Front-End Mentor with the objective of creating a Ticket Generator Page for a submition on Coding Conf

The Project was created using HTML, CSS and JavaScript. The input fields, such as name, email, github account and image were saved in the Local Storage and used to generate the final Ticket.
