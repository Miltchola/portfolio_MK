Age Calculator App

A React.js Application for a Age Calculator

About the Project

Developed during the Full Stack course, this project aimed to create an application using React.js to calculate the elapsed time (in days, months, or years) since a specific date provided by the user.

The project was implemented with React.js, Vite, and JavaScript.
