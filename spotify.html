Spotify Website

A Recreation of Spotify Website with HTML, CSS and JavaScript.

About the Project

This project was developed during the completion of a front-end course on an online study platform called Alura.

During the course, I was taught on how to use HTML, CSS and JavaScript to create a website from the beginning. The course also taught how to work with APIs and Backend to pull the information to the webpage. On the last class, I was also introduced to React.js.
